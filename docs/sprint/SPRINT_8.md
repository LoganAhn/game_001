## 작업 항목

### 1. 반응형 디자인
- [ ] 데스크탑 (1200px+): 전체 레이아웃
- [ ] 태블릿 (768px~1199px): 약간 축소된 테이블
- [ ] 모바일 (767px 이하): 세로 레이아웃, 터치 최적화
- [ ] 카드/버튼 크기 미디어 쿼리 대응
- [ ] 터치 이벤트 최적화 (슬라이더, 버튼)

### 2. 게임 완결 처리
- [ ] 플레이어 탈락 시 시각적 처리 (좌석 흐리게, "탈락" 표시)
- [ ] 게임 종료 화면: 승리/패배 결과, 통계 (핸드 수, 최고 핸드 등)
- [ ] "새 게임 시작" 버튼
- [ ] 게임 시작 화면 (환영 메시지, 시작 버튼)

### 3. 설정 메뉴
- [ ] 사운드 on/off 토글
- [ ] 볼륨 슬라이더
- [ ] 애니메이션 속도 조절 (빠르게/보통/느리게)
- [ ] 설정 아이콘 (톱니바퀴) + 드롭다운/모달

### 4. UX 개선
- [ ] 핸드 랭킹 참조 표 (도움말 버튼 → 팝업)
- [ ] 현재 핸드 강도 힌트 (초보자용, 토글 가능)
- [ ] 이전 핸드 결과 로그
- [ ] 키보드 단축키 (F=Fold, C=Check/Call, R=Raise)

### 5. 버그 수정 및 밸런스
- [ ] 엣지 케이스 테스트: 올인 vs 올인, 다중 사이드팟, 2인 헤즈업
- [ ] AI 밸런스 조정: 파라미터 미세 튜닝
- [ ] 블라인드 증가 속도 밸런스
- [ ] 메모리 누수 체크 (이벤트 리스너 정리)

### 6. 빌드 및 배포 준비
- [ ] `npm run build` 정상 동작 확인
- [ ] 번들 사이즈 확인
- [ ] 크로스 브라우저 테스트 (Chrome, Firefox, Safari, Edge)
